我的个人网站简述
***
- 采用欧美风格设计开发，基于 Bootstrap 框架的静态网页。
- 新增了简历下载功能和 Formspree 私信留言功能, 可以在上面给我留言私信，我会收到的。
- 服务器使用的是免费的 Github Pages， 速度和性能上会比收费的慢。
- 第一次打开会比较慢，所以我加了个开头动画；但因为是静态网页，第二次打开如果浏览器缓存还在会比较快。
- 手机和电脑显示会出于适配原因而看起来稍微不一样。
- 性能优化方面：
    + 我压缩了 html, css, js 的代码以减少传输的载荷。
    + 使用 JS 代码让所有图片懒加载。
    + 压缩图片和降低图片质量。
- 如果打开太慢，可以刷新一下或者关闭页面重新打开。也可以换个浏览器打开，换个网络。当然也有可能是服务器挂了，那就换个时间段打开。
